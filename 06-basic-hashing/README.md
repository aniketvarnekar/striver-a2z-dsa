# 05 — Basic Hashing

## [Problem 1: Count Frequency of Each Element in the Array](Problem01.java)

**Problem Statement:**  
Given an array, find the number of occurrences of each element in the array.

### Example 1  
**Input:**  
`arr[] = {10, 5, 10, 15, 10, 5}`  
**Output:**  
`10 3`  
`5 2`  
`15 1`  
**Explanation:**  
10 occurs 3 times in the array.  
5 occurs 2 times in the array.  
15 occurs 1 time in the array.

### Example 2  
**Input:**  
`arr[] = {2, 2, 3, 4, 4, 2}`  
**Output:**  
`2 3`  
`3 1`  
`4 2`  
**Explanation:**  
2 occurs 3 times in the array.  
3 occurs 1 time in the array.  
4 occurs 2 times in the array.

## [Problem 2: Find the Highest and Lowest Frequency Element](Problem02.java)

**Problem Statement:**  
Given an array of size `N`, find the elements with the highest and lowest frequency.

### Example 1  
**Input:**  
`array[] = {10, 5, 10, 15, 10, 5}`  
**Output:**  
`10 15`  
**Explanation:**  
The frequency of `10` is `3`, which is the highest.  
The frequency of `15` is `1`, which is the lowest.

### Example 2  
**Input:**  
`array[] = {2, 2, 3, 4, 4, 2}`  
**Output:**  
`2 3`  
**Explanation:**  
The frequency of `2` is `3`, which is the highest.  
The frequency of `3` is `1`, which is the lowest.
