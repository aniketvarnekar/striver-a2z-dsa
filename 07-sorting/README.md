# 07 — Sorting

## [Problem 1: Selection Sort](Problem01.java)

**Problem Statement:**
Given an array, sort it in ascending order using Selection Sort.

### Example 1
**Input:**
`arr[] = {13, 46, 24, 52, 20, 9}`

**Output:**
`9 13 20 24 46 52`

**Explanation:**
Selection Sort repeatedly selects the smallest element from the unsorted portion and swaps it into the next position.

### Example 1
**Input:**

## [Problem 3: Insertion Sort](Problem03.java)

**Problem Statement:**
Given an array, sort it in ascending order using Insertion Sort.

### Example 1
**Input:**
`arr[] = {7, 4, 1, 5, 3}`

**Output:**
`1 3 4 5 7`

**Explanation:**
Insertion Sort builds a sorted portion of the array by inserting each new element into its correct position.

## [Problem 4: Merge Sort](Problem04.java)

**Problem Statement:**
Given an array, sort it in ascending order using Merge Sort.

### Example 1
**Input:**
`arr[] = {3, 2, 8, 5, 1, 4, 23}`

**Output:**
`1 2 3 4 5 8 23`

**Explanation:**
Merge Sort divides the array into halves, sorts each half, and merges them back in sorted order.

## [Problem 5: Recursive Bubble Sort](Problem05.java)

**Problem Statement:**
Given an array, sort it in ascending order using a recursive Bubble Sort approach.

### Example 1
**Input:**
`arr[] = {13, 46, 24, 52, 20, 9}`

**Output:**
`9 13 20 24 46 52`

**Explanation:**
This is Bubble Sort performed recursively: after pushing the largest element to the end in a pass, the algorithm recurses on the remaining unsorted portion.

## [Problem 7: Quick Sort](Problem07.java)

**Problem Statement:**
Given an array, sort it in ascending order using Quick Sort.

### Example 1
**Input:**
`arr[] = {4, 1, 7, 9, 3}`

**Output:**
`1 3 4 7 9`

**Explanation:**
Quick Sort selects a pivot, partitions the array around it, and recursively sorts the left and right partitions.
